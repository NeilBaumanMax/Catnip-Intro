### 希望学习的地方

* 企业官网式大 Banner 首屏。
* 清晰的产品分类展示。
* 工业科技类 B2B 网站的信息结构。
* 首页突出主营产品、应用场景和公司能力。
* 产品卡片简洁，适合外贸和企业客户快速浏览。
* 关于我们页面突出技术路线、研发能力和落地场景。
* 联系我们页面清晰展示电话、地址和留言表单。
* 页面整体要专业、可信、偏科技型企业风格。

### 不希望照搬的地方

* 不复制参考网站文案。
* 不复制参考网站图片。
* 不复制参考网站 logo。
* 不完全复刻参考网站布局。
* 只参考企业官网的信息组织、视觉节奏和产品展示方式。
* 保留 Catnipent 自身的 AI+硬件、本地 Agent、端侧智能特色。

### 色调偏好

| 用途  | 颜色                                  |
| --- | ----------------------------------- |
| 主色  | 深蓝 / 科技蓝，例如 #0F3D7A 或 #1A56DB       |
| 辅助色 | 青蓝 / 电光蓝，例如 #00AEEF                 |
| 背景色 | 白色、浅灰、冷灰蓝，例如 #F6F8FB                |
| 强调色 | 绿色或橙色少量点缀，用于“本地部署 / 智能控制 / 安全可信”等标签 |

### 风格关键词

简洁、科技感、专业、工业科技、AI 硬件、B2B 企业官网、端侧智能、本地部署、安全可信、软硬件一体化、现代、可靠、可交付。

## 七、前端改版重点

### 首页重点

首页需要突出：

1. Catnipent 是一家本地 Agent 软硬件一体化公司。
2. 核心项目是“万物有灵”。
3. 主推本地 Agent 智能板卡 / 龙虾派端侧 AI 控制主板。
4. 技术路线是 Agent Runtime + Edge AI + Hardware。
5. 应用场景包括学校、园区、企业知识库、工业控制、机器人控制、智能办公和创客教育。
6. 强调本地部署、数据不出域、断网可用、长期运行、软硬件一体化交付。

# design-input

前端设计资料包目录。

## 目录结构

```
design-input/
├─ DESIGN_BRIEF.md    # 设计概要（填写后提供设计指导）
├─ README.md          # 本文件
├─ reference/         # 参考网站截图
├─ logo/              # Logo 文件（png/svg）
├─ products/          # 产品图片
├─ cases/             # 案例图片
└─ company/           # 公司相关图片（外景、团队、资质等）
```

## 使用方式

1. 填写 `DESIGN_BRIEF.md` 中的各节内容。
2. 将参考网站截图放入 `reference/`。
3. 将 logo 文件放入 `logo/`。
4. 将产品图片放入 `products/`。
5. 将案例图片放入 `cases/`。
6. 将公司相关图片放入 `company/`。

全部就绪后启动前端改版 Phase。
